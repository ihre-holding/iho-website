# iho-website

Webauftritt der [Ihre Holding GmbH](https://www.ihre-holding.de) – familiengeführte Beteiligungsgesellschaft für ambulante Pflegedienste in Bayern.

## Inhalt

- Startseite: Nachfolge & Übergabe, Beratung & Begleitung, Kontakt
- Impressum und Datenschutz

## Entwicklung

```bash
npm install
npm run dev
```

Öffnen: http://localhost:4321/

## Build & Deploy

```bash
npm run build
```

Deployment über GitHub Actions auf `main` → GitHub Pages (`www.ihre-holding.de`).

## Stack

- [Astro](https://astro.build/) 5
- Statisches Site-Deployment
